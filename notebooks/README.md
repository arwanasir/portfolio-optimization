# Notebooks Directory

This directory contains Jupyter notebooks for different stages of the analysis.

## Notebook Files

- `01_data_extraction.ipynb` - Fetch and clean data from YFinance
- `02_eda.ipynb` - Exploratory Data Analysis and visualization
- `03_arima_modeling.ipynb` - ARIMA/SARIMA model implementation
- `04_lstm_modeling.ipynb` - LSTM neural network implementation
- `05_portfolio_optimization.ipynb` - Efficient Frontier generation
- `06_backtesting.ipynb` - Strategy validation and comparison

## Usage

Launch Jupyter from the project root:

```bash
jupyter notebook notebooks/
```
